## ⚙️ 如何使用

1. **选择协议**：选择您要使用的协议（VMess、VLESS、SS或Trojan）。
2. **选择国家**：从“国家配置”中选择您喜欢的国家。
3. **下载配置**：选择协议和国家后，点击相应链接下载配置文件。
4. **在V2Ray中使用配置**：将下载的配置导入到V2Ray应用中并开始使用。

配置每15分钟更新一次，确保用户始终可以访问到最新的链接。
